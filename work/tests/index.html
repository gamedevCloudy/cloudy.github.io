Forest But Banana

A pomodoro app that helps Banana enthusiasts be productive.

- Aayush Chaudhary

The Goal

Initially, the goal was very simple, request Forest team to add bananas.
But then, I know how to code. So, I decided to cultivate my own plantations. Becuase, you know, I was growing it myself, thus it is a plantation.

Why Banana? To answer this here's Jon's POV:

"Its Banana, what else do you want?" - Jon

The Task

Thus, I decided to crate a forest clone, but it must have bananas. Only BANANAS.

Requirements: - BANANAS - Pomodoro Timer: like forest, there should be a round spinny rotating time indication thingy - Custom Timing: user can set their custom timers - Half baked low qual UX

Action

Since, we don't know much about developing apps classicaly, we stick to my area of expertise, Unity and C#.

Now, Unity is a game engine. But I wanted to make use of its UI system to create the user interface. And logic is very simple, so not much to worry here.

Result

The final app has basic functionalities. - User can create custom timers. - User can optionally listen to White Noise(Rainforest bgm) - User can see for how many minutes they have focused till now. - User can restart.

Also, - it has interstitial ads and banner ads.
